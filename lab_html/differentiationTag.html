Differentiation tag

Tag Similarity Differentiation
< hr > & < br > used to insert a line
  • < br > is used to insert a line break which means the text/image following the tag will be moved to the next line while.
  • < hr > is used to insert horizontal line that can divide the information into sections.
< ul > & < ol > used to show list
  • < ul > or bulleted list is an unordered list which means that the items in this list are not placed in a specific order.
  • < ol > or numbered list ,the items are placed in the specific order and prefaced with a number.
< select >&< checkbox >&< radio button > used to make decision
  • The < select > element is used to create a drop-down list.()
  • The < checkbox > element is used to create a checkbox.(independent from each other)
  • The < radio button > element is used to create a radio button.(supposed to choose one of the options)
< div > & < span > they are section
  • The div tag is known as Division tag. The div tag is used in HTML to make divisions of content on the web page
  • The HTML span element is a generic inline container for inline elements and content. It used to group elements for styling purposes.
    A better way to use it when no other semantic element is available.The span tag is very similar to the div tag, but div is a block-level tag and span is an inline tag.
< script > & < style > used to define
  • The < script > tag is used to define a client-side script (JavaScript).
  • The < style > tag is used to define style information for an HTML document.
